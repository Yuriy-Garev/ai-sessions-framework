# Sessions History Detailed

Store archived **detailed** session files here.

## Purpose
This folder is the **freezing** archive level.
Use it for deep historical recovery.

## Rules
- one archived session per file
- do not append to a giant log
- filenames must follow:
  - `YYYY-MM-DD__session-goal-detailed.md`
- keep files factual and detailed
- listing filenames is cheap navigation
- opening file contents requires user approval unless explicitly allowed
