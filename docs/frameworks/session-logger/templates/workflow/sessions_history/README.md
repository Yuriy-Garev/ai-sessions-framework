# Sessions History

Store archived **summary** session files here.

## Purpose
This folder is the **cold** archive level.
Use it for lightweight historical recovery.

## Rules
- one archived session per file
- do not append to a giant log
- filenames must follow:
  - `YYYY-MM-DD__session-goal-summary.md`
- keep files concise and restart-oriented
- listing filenames is cheap navigation
- opening file contents requires user approval unless explicitly allowed
