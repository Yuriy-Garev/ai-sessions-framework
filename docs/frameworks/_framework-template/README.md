# Framework Template

Copy this folder to create another reusable framework.

## Fill these sections

### Purpose
What problem the framework solves.

### Trigger conditions
When Codex or the user should use it.

### Non-goals
What this framework is not for.

### Owned files
Which files or directories this framework controls.

### Procedure
Step-by-step operating rules.

### Integration path
How it connects to:
- `AGENTS.md`
- one or more repo-local skills
- any scripts or templates
