# Source Basis

This skill distills practical CLI help guidance from these standards, guides, and references:

- Command Line Interface Guidelines: https://clig.dev/
- GNU Coding Standards for `--help`, `--version`, and long-option conventions: https://www.gnu.org/prep/standards/
- POSIX Utility Syntax Guidelines and rationale: https://pubs.opengroup.org/onlinepubs/
- docopt documentation: https://docopt.readthedocs.io/
- Git help and CLI conventions: https://git-scm.com/docs/
- Cobra user guide: https://github.com/spf13/cobra/
- Click documentation: https://click.palletsprojects.com/
- clap documentation: https://docs.rs/clap/
- Python argparse documentation: https://docs.python.org/3/library/argparse.html
- Kubernetes kubectl reference structure: https://kubernetes.io/docs/reference/kubectl/
- Linux man-pages guide: https://man7.org/linux/man-pages/man7/man-pages.7.html

Use these links when a user asks for standards justification, ecosystem precedent, or deeper reading. Do not load this reference for routine CLI help rewriting unless source attribution matters.
